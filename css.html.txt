          <p>Use <code>&lt;small&gt;</code> tag or <code>.small</code> class to make a paragraph smaller, or make a paragraph stand out by adding <code>.lead</code> class.</p>

          <h3>Blockquotes</h3>
          <p>Wrap <code>&lt;blockquote&gt;</code> around any HTML as the quote. For straight quotes, we recommend a <code>&lt;p&gt;</code>.</p>
          <div class="code-snippet">
            <div class="code-preview">
              <blockquote>
                <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Integer posuere erat a ante.</p>
              </blockquote>
            </div>

<pre class="line-numbers no-name"><code class="language-markup">
&lt;blockquote&gt;
  &lt;p&gt;Lorem ipsum dolor sit amet, consectetur adipiscing elit. Integer posuere erat a ante.&lt;/p&gt;
&lt;/blockquote&gt;
</code></pre>
          </div>

          <h4>Naming a source</h4>
          <p>Add a <code>&lt;footer&gt;</code> for identifying the source. Wrap the name of the source work in <code>&lt;cite&gt;</code>.</p>
          <div class="code-snippet">
            <div class="code-preview">
              <blockquote>
                <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Integer posuere erat a ante.</p>
                <footer>Someone famous in <cite title="Source Title">Source Title</cite></footer>
              </blockquote>
            </div>

<pre class="line-numbers no-name"><code class="language-markup">
&lt;blockquote&gt;
  &lt;p&gt;Lorem ipsum dolor sit amet, consectetur adipiscing elit. Integer posuere erat a ante.&lt;/p&gt;
  &lt;footer&gt;Someone famous in &lt;cite title=&quot;Source Title&quot;&gt;Source Title&lt;/cite&gt;&lt;/footer&gt;
&lt;/blockquote&gt;
</code></pre>
          </div>

          <h3>Header</h3>
          <p>You can see sample header almost in every section of this documentation. The header of this section uses following code:</p>
<pre class="line-numbers"><code class="language-markup">
&lt;header class=&quot;section-header&quot;&gt;
  &lt;span&gt;Content decoupling&lt;/span&gt;

  &lt;p&gt;Split your documentation page to different parts and describe each of those inside a section tag.&lt;/p&gt;
&lt;/header&gt;
</code></pre>

          <h4>Alignment</h4>
          <p>By default, the header is center aligned. But you can easily change the text alignment by using <code>.text-left</code> or <code>.text-right</code> with header tag.</p>
          <div class="code-snippet">
            <div class="code-preview">
              <header class="section-header text-left">
                <span>Content decoupling</span>
                <h2>Sections</h2>
                <p>Split your documentation page to different parts and describe each of those inside a section tag.</p>
              </header>
            </div>

<pre class="line-numbers no-name"><code class="language-markup">
&lt;header class=&quot;section-header text-left&quot;&gt;
  &lt;span&gt;Content decoupling&lt;/span&gt;
  &lt;h2&gt;Sections&lt;/h2&gt;
  &lt;p&gt;Split your documentation page to different parts and describe each of those inside a section tag.&lt;/p&gt;
&lt;/header&gt;
</code></pre>
          </div>


          <h3 id="sec-sections-bg">Background</h3>
          <p>You have a plenty of option to modify background of a section.</p>

          <h4>Color</h4>
          <p>By default, background-color is white, but you can use following classes with a <code>section</code> tag to modify its background-color.</p>

<pre class="simple-view no-copy"><code class="language-markup">
&lt;section class=&quot;bg-gray-light&quot;&gt;
  ...
&lt;/section&gt;
</code></pre>

          <ul class="color-palette-circular">
            <li class="bg-purple">.bg-purple</li>
            <li class="bg-green">.bg-green</li>
            <li class="bg-blue">.bg-blue</li>
            <li class="bg-orange">.bg-orange</li>
            <li class="bg-red">.bg-red</li>
            <li class="bg-dark">.bg-dark</li>
            <li class="bg-dark-light">.bg-dark-light</li>
            <li class="bg-dark-lighter">.bg-dark-lighter</li>
            <li class="bg-dark-lightest">.bg-dark-lightest</li>
            <li class="bg-gray txt-gray">.bg-gray</li>
            <li class="bg-gray-light txt-gray">.bg-gray-light</li>
            <li class="bg-gray-lighter txt-gray">.bg-gray-lighter</li>
            <li class="bg-gray-lightest txt-gray">.bg-gray-lightest</li>
          </ul>

          <p>Also, you can always change background-color using inline style:</p>
<pre class="simple-view no-copy"><code class="language-markup">
&lt;section style=&quot;background-color: #123456&quot;&gt;
  ...
&lt;/section&gt;
</code></pre>

          <h4>Image</h4>
          <p>Easily add a background-image to a section using inline style:</p>
<pre class="simple-view no-copy"><code class="language-markup">
&lt;section style=&quot;background-image: url(path/to/image.png)&quot;&gt;
  ...
&lt;/section&gt;
</code></pre>
          
          <p>Or take advantage of your <code>assets/css/custom.css</code> file and make a class for each background-image:</p>
          <div class="code-snippet">
<pre class="line-numbers no-copy"><code class="language-css">
.bg-img-sec-about {
  background-image: url(../img/bg-about.png);
}
</code></pre>
<pre class="line-numbers no-copy"><code class="language-markup">
&lt;section class=&quot;bg-img-sec-about&quot;&gt;
  ...
&lt;/section&gt;
</code></pre>
          </div>


          <h4>Video</h4>
          <p>You can easily add a video to background of sections.</p>
          <div class="code-splitted">
            <div class="code-group col-md-7">
<pre class="line-numbers"><code class="language-markup">
&lt;section&gt;
  &lt;video class=&quot;bg-video&quot; autoplay loop&gt;
    &lt;source src=&quot;assets/img/video.mp4&quot; type=&quot;video/mp4&quot;&gt;
    &lt;source src=&quot;assets/img/video.webm&quot; type=&quot;video/webm&quot;&gt;
  &lt;/video&gt;
&lt;/section&gt;
</code></pre>
            </div>

            <div class="code-preview col-md-5">
              <section>
                <video class="bg-video" autoplay loop>
                  <source src="assets/img/video.mp4" type="video/mp4">
                  <source src="assets/img/video.html" type="video/webm">
                </video>
              </section>
            </div>
          </div>

          <h3>Overlay</h3>
          <p>It's always a good practice to accompany background images and videos with an overly class to make a better contrast between background and texts. You have <code>.overlay-black</code> and <code>.overlay-white</code> classes to use with sections. Also, you can increase or decrease their opacity by adding <code>.overlay-intense</code> and <code>.overlay-pale</code> respectively.</p>
